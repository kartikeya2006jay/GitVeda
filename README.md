## 🚀 Quick Start

### 1. Initialize
```bash
git clone https://github.com/kartikeya2006jay/GitVeda.git
cd GitVeda
```

### 2. Prepare Environment
```bash
npm install
cp .env.example .env
```
*Fill in your Firebase credentials in the `.env` file.*

### 3. Launch Core
```bash
npm run dev
```
